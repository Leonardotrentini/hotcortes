1️⃣ CRIAR REPOSITÓRIO NO GITHUB
   ─────────────────────────────────────────────────────
   • Acesse: https://github.com/new
   • Nome: corteshot
   • Clique em "Create repository"


2️⃣ ENVIAR CÓDIGO PARA GITHUB
   ─────────────────────────────────────────────────────
   Abra o PowerShell e execute:

   cd "C:\Users\Leonardo trentini\Desktop\ferramentahot"
   git init
   git add .
   git commit -m "Deploy CortesHot"
   git branch -M main
   git remote add origin https://github.com/SEU_USUARIO/corteshot.git
   git push -u origin main

   ⚠️ Substitua SEU_USUARIO pelo seu usuário do GitHub


3️⃣ FAZER DEPLOY NA VERCEL
   ─────────────────────────────────────────────────────
   • Acesse: https://vercel.com
   • Faça login com GitHub
   • Clique em "Add New Project"
   • Selecione o repositório "corteshot"
   • Clique em "Deploy"
   • Aguarde 2-5 minutos
   • Pronto! 🎉
